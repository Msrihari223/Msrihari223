REST API Endpoints

Your Flask server runs on http://localhost:5000. Test with Postman or use the JavaScript Fetch API below.

GET/api/health
// Response { "status": "online", "model": "loaded" | "demo_mode", "endpoints": ["/api/predict", "/api/batch"] }
POST/api/predict
// Request Body (JSON) { "text": "message to analyze", "platform": "Instagram" } // Response { "label": "SAFE" | "WARNING" | "DANGER", "confidence": 0.91, "confidence_pct": "91%", "categories": ["Harassment"], "action": "Alert moderator...", "source": "model" | "demo" }
POST/api/batch
// Request Body (JSON) { "messages": ["msg1", "msg2", "msg3"] } // Response { "results": [ { "text": "msg1", "label": "SAFE", "confidence": "95%" }, { "text": "msg2", "label": "DANGER", "confidence": "88%" } ], "total": 2 }
JavaScript Fetch Example
// How this frontend calls the Flask API const response = await fetch("http://localhost:8080/api/predict", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ text: "your message here", platform: "Instagram" }) }); const data = await response.json(); console.log(data.label, data.confidence_pct);
